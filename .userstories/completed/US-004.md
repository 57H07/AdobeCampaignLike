## Implementation Summary
`TemplateStorageOptions` is a strongly-typed options class bound from the `TemplateStorage` section in `appsettings.json` with a single `RootPath` property (default: empty string). `TemplateStorageStartupValidator` implements `IHostedService`; its `StartAsync` runs before any web requests are served and throws `InvalidOperationException` (halting startup) if `RootPath` is null/whitespace, if the directory does not exist, or if a temp file write fails (indicating insufficient permissions). Errors are logged via `IAppLogger` before throwing. The service is registered via `AddHostedService<TemplateStorageStartupValidator>()` in the Infrastructure `AddInfrastructure()` extension, and `appsettings.json` has the new `TemplateStorage:RootPath` top-level section defaulting to `"templates"`. 12 unit tests cover all failure paths and the happy path.
