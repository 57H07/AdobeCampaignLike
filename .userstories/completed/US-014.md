## Implementation Summary
`DocxPlaceholderReplacer` is a sealed class in `CampaignEngine.Infrastructure.Rendering` with a single public method `ReplaceScalars(WordprocessingDocument doc, Dictionary<string, string> data)`. It traverses the main document body, all HeaderParts, and all FooterParts, finds every `Text` node that contains `{{`, then applies a compiled regex `\{\{\s*(?<key>[^{}\s]+)\s*\}\}` to replace matched tokens. Values are XML-escaped via `SecurityElement.Escape` (handles `&`, `<`, `>`, `"`, `'`), and missing keys silently produce an empty string replacement. The static helper `XmlEscape(string)` is public for direct unit testing. All 25 new tests pass; the method is safe to call after `DocxRunMerger.MergeRuns` has consolidated fragmented runs.
