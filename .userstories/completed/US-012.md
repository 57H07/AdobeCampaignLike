## Implementation Summary
Updated the template creation page to implement a conditional UI toggle driven by the channel selector. The channel dropdown defaults to Email, showing the HTML body-path editor. Selecting "Letter" hides the body-path input and reveals a file upload input restricted to `.docx` files via the `accept` attribute. A JavaScript IIFE wires up the `change` event on the channel selector to toggle visibility of both sections. Client-side validation prevents form submission for Letter when no file is selected or the file is not `.docx`. Server-side validation in `OnPostAsync` mirrors this logic: BodyPath is checked for Email/SMS, and the DOCX file upload is checked for Letter. The `IFormFile? DocxFile` property was added to `CreateTemplateInputModel`; for Letter the BodyPath is auto-derived from the uploaded filename.
