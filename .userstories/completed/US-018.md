## Implementation summary

### Architecture

New `IDocxPlaceholderParserService` interface (Application layer) wraps the existing `DocxPlaceholderParser` (Infrastructure). The controller buffers the DOCX stream, calls `CreateAsync`/`UpdateAsync` (file persisted), then runs manifest validation and populates `TemplateDto.Warnings`.
